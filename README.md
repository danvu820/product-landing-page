# Velora pillow landing page

A lightweight static website for a premium pillow product brand, inspired by modern editorial DTC layouts.

## Pages

- `/` – primary marketing landing page
- `/contact/` – support contact form page
- `/tracking/` – order tracking entry page

## Features

- Scroll-reveal animations for key page elements
- Mobile-first responsive layout
- GitHub Pages compatible static routing

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy to GitHub Pages

1. Push to GitHub.
2. In **Settings → Pages**, set **Source** to **GitHub Actions**.
3. Push to your deployment branch (configured in workflow) or run it manually.
